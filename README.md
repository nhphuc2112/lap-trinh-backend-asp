## Getting Started
- Install:
```sh
git clone https://github.com/nhphuc2112/lap-trinh-backend-asp.git
```
- Mở port 5076.
- Mở cmd:
  ```sh
  cd /project
  dotnet run
  ```
- Hoặc mở bằng Visual Studio với .sln -> Run with Http
- Browser: http://localhost:5076
